Installation¶

Using PIP

$ pip install django-stories

or download the app here

$ python setup.py install

Add stories to your settings INSTALLED_APPS

INSTALLED_APPS = (
    ...
    'stories',
    ...
)

Run syncdb

$ ./manage.py syncdb
